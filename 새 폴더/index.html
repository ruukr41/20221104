5-1.display요소와 각 요소에 대한 설명

  1. display: none; 화면에서 사라짐. 크기 자체도 차지하지 않음

  2. display: block; 일반적으로 설정하지 않아도 div가 갖게되는 기본값입. (태그에 따라 기본값이 다를 수 있음.) 기본적으로 width 가 자신의 컨테이너의 100% 가 되게끔 함. 쉽게 말해, 가로 한 줄을 다 차지하게 됨.

  3. display: inline; 컨텐츠를 딱 감쌀정도의 크기만 갖게됨. block태그와 다르게 줄바꿈이 되지 않고, 반드시 컨텐츠를 감싸게 되고, 크기를 변화시킬 수 없음.

  4. display: inline-block; inline과 block의 특성을 합쳐놓은 속성임. 기본적으로는 inline의 속성을 지니고 있지만, 임의로 크기를 바꿔줄 수 있음.






5-2.position요소와 각 요소에 대한 설명

position 속성을 별도로 지정해주지 않으면 기본값인 static이 적용됩니다. 이 말은 요소들이 HTML에 작성된 순서 그대로 브라우저 화면에 표시가 된다는 것을 뜻하며, 따라서 top, left, bottom, right 속성값은 position 속성이 static일 때는 무시됩니다.




  1. position: fixed
    position 속성을 fixed로 지정하면 이렇게 요소를 항상 고정된(fixed) 위치에 배치할 수 있습니다. 이게 가능한 이유는 fixed 속성값의 배치 기준이 자신이나 부모 요소가 아닌 뷰포트(viewport), 즉 브라우저 전체화면이기 때문인데요. top, left, bottom, right 속성은 각각 브라우저 상단, 좌측, 하단, 우측으로 부터 해당 요소가 얼마나 떨어져있는지를 결정합니다.

  2. position: relative
    position 속성을 relative로 설정하게 되면, 요소를 원래 위치에서 벗어나게 배치할 수 있게 됩니다. 요소를 원래 위치를 기준으로 상대적(relative)으로 배치해준다고 생각하시면 이해가 쉬울 것 같은데요. 요소의 위치 지정은 top, bottom, left, right 속성을 이용해서, 요소가 원래 위치에 있을 때의 상하좌우로 부터 얼마나 떨어지게 할지를 지정할 수 있습니다.

  3. position: absolute
    position 속성이 absolute일 때 해당 요소는 배치 기준을 자신이 아닌 상위 요소에서 찾습니다. 부모 요소(가장 가까운 상위 요소)를 기준으로 top, left, bottom, right 속성을 적용. 어떤 요소의 display 속성을 absolute로 설정하면, 부모 요소의 display 속성을 relative로 지정해주는 것이 관례.

  4. position: sticky
    부모요소의 상 하에 스크롤에 따라 붙는다. 말그대로 스크롤링에 따라 스티커처럼 끈적하게 붙는다.

6.animation [css는 여기가 고비이며 마지막 page임]

animation에는 크게 볼 떄 두가지 속성이 필요함. '시간'과 '상태'임 시간과 관련 된 속성들은 animation에서 고유적으로 사용 됨. 상태와 관련된 속성들은 수업중 계속 써왔음 (border,color,width,height 등등..)

animation속성에 활용되는 태그는 너무 많기 때문에 구글링으로 예제들을 보는게 좋을 듯 하다.
